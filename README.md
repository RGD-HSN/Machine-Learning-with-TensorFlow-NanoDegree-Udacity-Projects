## Overview
This repository contains projects completed as part of the **Machine Learning with TensorFlow Nanodegree**. It demonstrates practical experience in machine learning and deep learning, including data preprocessing, feature engineering, model building, and evaluation using Python and TensorFlow.  

These projects showcase real-world applications of supervised and unsupervised learning, neural networks, and data-driven decision-making.

---

## Projects

### Finding Donors
- **Objective:** Predict potential donors using machine learning.  
- **Techniques:** Feature selection, Random Forest algorithm, model evaluation (accuracy & F-score).  
- **Files:** `finding_donors/` folder contains all code, dataset, and results.

### Image Classifier
- **Objective:** Build a model to classify images into categories.  
- **Techniques:** Convolutional Neural Networks (CNNs), data augmentation, model training and evaluation.  
- **Files:** `image_classifier/` folder contains all code, dataset links, and results.

### Identify Customer Segments
- **Objective:** Segment customers based on behavior and demographics.  
- **Techniques:** K-Means clustering, feature engineering, cluster analysis.  
- **Files:** `identify_customer_segments/` folder contains all code, dataset, and visualizations.
